# Task-5
HTML and CSS project featuring a Laundry Services landing page with a navigation bar, hero section, service description, image, and booking button.

This project demonstrates the creation of a Laundry Services Hero Section using HTML and CSS. It includes a navigation bar with a logo, navigation links, and a username button, along with a responsive hero section containing a heading, description, call-to-action button, and a laundry service image. The layout is built using CSS display properties and viewport units without using Flexbox.

## Features

- Navigation Bar
- Logo and Username Section
- Inline Display Layout
- Hero Section
- Responsive Design using Viewport Units (vw, vh)
- Styled Button
- Image Integration
- Clean Typography
- No Flexbox

## Files

- index.html
- style.css

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Make sure index.html and style.css are in the same directory.
4. Open index.html in any web browser.
